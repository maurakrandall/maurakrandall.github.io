# Connection Tools
**Quick team health checks and daily connection rituals**

## Purpose
These lightweight apps help teams stay connected through brief, async check-ins. Use them for daily standups, temperature checks, and quick trust-building exercises that take seconds to complete but create lasting connection.

**Quick team temperature reading**

## When to Use Connection Tools

**Daily:** Daily Drop keeps everyone aligned without meetings

**Weekly:** Temp Check gives you early warning when something's off

**Monthly:** Trust Check tracks team health over time

**As needed:** Warm-Up makes any meeting more human

---

## Design Philosophy

All Connection Tools follow these principles:
- **Fast:** Complete in under 2 minutes
- **Async-first:** No scheduling required
- **Low stakes:** Not everything needs to be a production
- **Pattern recognition:** Individual responses matter less than trends
- **Action-oriented:** Results should prompt conversation, not just observation
